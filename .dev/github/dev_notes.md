# IT 140 GitHub Development Notes

## GitHub Repositories

### Course Repository

Every course that uses GitHub has a main course repository. For courses that have just one repository, the main course repository is the only repository for the course. Activity materials will be in subdirectories of this repository.

The URL format for the main course repository is `https://github.com/GC-STEM/crs000`, where `crs000` is the course code.

The CS program main course repository master template is `https://github.com/GC-STEM/cs000`.

### Activity Repositories

Courses that have multiple repositories will have a main course repository and two or more activity repositories. The main course repository `README.md` will contain links to the activity repositories

The URL format for activity repositories is `https://github.com/GC-STEM/crs000-activity`, where `crs000` is the course code and `activity` is the to-be-determined activity designator.

The CS program activity repository master template is `https://github.com/GC-STEM/cs000-activity`.
