# Selecting a Professional GitHub Username

Your **GitHub username** is your public identity on GitHub. Choose a username that you will be comfortable sharing professionally for many years.

Your username becomes part of your GitHub profile address, repository URLs, and other places where you use Git and GitHub. You may also include your GitHub profile on your résumé, CV, LinkedIn profile, personal website, or job applications. In technical and computational fields, employers may view your GitHub profile as part of your professional portfolio.

## GitHub Usernames Are Public

Anyone may see your GitHub username when you:

* Share your GitHub profile
* Create a public repository
* Contribute to a public repository
* Participate in an open-source project
* Share a repository with an instructor, tutor, or employer
* Include your GitHub profile on a résumé, CV, LinkedIn profile, personal website, or job application

Making your GitHub profile private does not hide your username or make public repositories private.

## Characteristics of a Good GitHub Username

Consider how Petey Penmen, the SNHU mascot, might choose a professional GitHub username.

A good username is:

* **Professional.** Use your real or preferred professional name when possible.
* **Clear.** Make it easy to read, spell, and recognize.
* **Brief.** Keep it reasonably short.
* **Timeless.** Choose a username that will still be appropriate after graduation.
* **Distinctive.** Select a version that distinguishes you from other GitHub users without revealing private information.

| Good Example   | Why It Works                                            |
| -------------- | ------------------------------------------------------- |
| `peteypenmen`  | Uses a full name and is easy to remember                |
| `petey-penmen` | Separates the names for readability                     |
| `ppenmen`      | Uses an initial and last name                           |
| `peteyjpenmen` | Adds a middle initial without using private information |

## If Your Preferred Username Is Taken

Your first choice may already belong to another GitHub user. Try professional variations in this order:

1. Add your middle initial or middle name
2. Use your first initial and last name
3. Add a hyphen between parts of your name
4. Reverse the order of your first and last names
5. Add a brief professional term only if it reflects a long-term interest or identity

For example, if `peteypenmen` is unavailable, Petey could try:

* `peteyjpenmen`
* `petey-j-penmen`
* `ppenmen`
* `penmen-petey`

Avoid adding random numbers just to make a username available. If numbers are necessary, use a combination that does not reveal personal information and will remain professional.

> [!TIP]
> Before creating your account, say the username aloud and imagine placing it on a résumé. Choose another option if it is difficult to explain, spell, or share professionally.

## Usernames to Avoid

Avoid usernames that include:

* Birth years or graduation years
* Student identification numbers
* Birth dates, phone numbers, addresses, or other private information
* Random numbers when a name-based alternative is available
* Slang, jokes, or internet memes
* Offensive or inappropriate language
* Nicknames that may appear unprofessional
* Temporary interests that you may outgrow

Examples to avoid:

* `peteypenmen2007`
* `coolguy99`
* `xXPeteyMasterXx`
* `gamergal123`

> [!IMPORTANT]
> Choose a username that you will be comfortable sharing professionally throughout college and your career.

## GitHub Username and Git Identity

Your GitHub username and Git author information serve different purposes.

* Your **GitHub username** identifies your GitHub account, such as `petey-penmen`.
* Your **Git author name** identifies you as the person who created a commit, such as `Petey Penmen`.
* Your **Git author email** helps GitHub associate your commits with your GitHub account.

You should normally use your real or preferred professional name—not your GitHub username—as your Git author name.

For example:

```text
GitHub username: petey-penmen
Git author name: Petey Penmen
```

A professional Git author name makes commit history easier for instructors, tutors, collaborators, and prospective employers to understand. It also helps reviewers connect your work with the name on your résumé or job application.

Using your GitHub username as your Git author name is acceptable when the username is already a clear version of your name. However, matching the two does not provide an additional technical benefit. GitHub normally uses the commit email address—not an exact match between the author name and username—to associate commits with an account.

Later, you will configure Git with your professional author name and the private `noreply` email address provided by GitHub.

For Petey Penmen, the recommended Git author name would be:

```text
Petey Penmen
```

## Recommendation

Choose a professional version of your real or preferred name whenever possible. If your first choice is unavailable, try a middle initial, an initial-and-last-name format, or another clear name-based variation.

For Petey Penmen, `petey-penmen` is a strong choice because it is easy to read, professional, and appropriate for both academic and future career use.
